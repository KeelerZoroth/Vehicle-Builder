## Installation
  Prerequisites: Node.js, npm  


  clone the repo: ``` git clone https://github.com/KeelerZoroth/Vehicle-Builder.git ```  

enter the folder: ``` cd Vehicle-Builder ```  

Install npm packages: ``` npm i ```
## Usage
  Run ``` npm run start ```
